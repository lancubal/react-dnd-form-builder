# React Drag & Drop Form Builder

A modern "No-Code" form builder utilizing:
- **React 18**
- **Zustand** for complex global state management.
- **@dnd-kit** for accessible drag-and-drop interactions.
- **Tailwind CSS** for styling.
- **Dynamic Registry Pattern** for scalable component rendering.

## Features
- Drag and drop elements to reorder.
- Add new elements from the sidebar.
- Edit properties (label, placeholder, required, options) dynamically.
- Centralized state management.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
